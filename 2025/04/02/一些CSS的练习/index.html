跟着 B 站写点 CSS

这学期在学 web 开发，想着自己写点东西来练习一下这项技能，前端工资不高但是如果自己会别人不会的应该还是能找碗饭吃的，所以空闲时就跟着 B 站上的视频写点简单的东西。

这并不耗费我什么精力，中午写不进去算法题，写点 CSS 也算比较养生。
由于我并不知道怎么单独展示一个独立的 html 标签(因为这个 hexo 框架会自动渲染)，所以使用 markdown 进行演示，所以每个文章只能有一个例子，请通过 tags 查询其他。
